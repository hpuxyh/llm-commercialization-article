# 大模型商业化 · 核心要点（交接文档）

> 用途：粘到新对话窗口当上下文，让下一轮直接接着聊，不用重述。
> 主旨：**互联网赚钱方式正在迁移，大模型本身不赚钱，平台靠"第二条腿"变现，但每条腿都要交"自我蚕食税"——亏的是豆包，赚的是字节。**

---

## 一、一句话主命题
大模型平台商业化不是"卖模型"，而是从**注意力经济 → 意图经济**：过去靠用户停留/浏览卖广告，未来靠理解意图、影响决策、完成交易来收广告费和佣金。

## 二、核心逻辑链（7 个模块）
1. **大模型本身亏钱**：C 端免费 + API 负毛利 + 训练无底洞。单看 token，收入只覆盖算力的 ~14%。
2. **钱的三级跳**：被搜索(SEO·搜索广告) → 被提及(GEO·AI 答案广告) → 被买单(Agent·抽佣)。**搜索广告几千亿大盘不会消失，会迁移成"AI 答案广告"**。
3. **用户为什么必走这条路**：用户要的不是信息，是"把事办成"；每代技术替用户多干一步（链接→答案→替你下单）。**只要更省事就会迁移**，钱跟着注意力走。
4. **四条变现腿**：订阅(面子) / API(走量) / 广告(里子·兜底) / agent 抽佣(胜负手)。全开仍填不平成本。
5. **自我蚕食税**：新生意吃老生意，每步都要交"左右互搏"税。**规律：谁的现有饭碗会被自己的 AI 吃掉，谁就慢。**
6. **最深真相·两本账**：豆包不是独立生意，是"入口期权"。账面亏，但为字节①防御基本盘②赋能增收③toB 云④期权——**亏的是豆包，赚的是字节**（同亚马逊靠 AWS、Google 靠广告）。
7. **多方博弈终局**：不是一家通吃，而是**分层分化**——免费层(带广告/抽佣)+付费层(去广告)+中立层(清流)三层并存。

**8% 怎么来**：agent 直接买丢了"逛"才有的 15% 投流广告，新"竞价加权"因 AI 答案信任天花板只能补回 ~3%，加 5% 基础佣金 = 8%。不是硬上限。

**市场≠捕获**：1 万亿市场（如客服）→ 平台真正能记进账的可能只剩 ~150 亿（替代率 × 定价折扣 × 分层份额三道折扣），且记在 toB 云，不是广告/抽佣。

## 四、核心概念词（自创/复用）
- **自我蚕食税**：新业务吃老业务的左右互搏成本。
- **入口期权**：大模型亏钱也要做，因为是下一代入口门票。
- **饭碗规律**：谁的现有饭碗会被自己的 AI 吃掉，谁就慢；谁背后有钱输血，谁就能熬。

## 五、各家定位（决策因素=现有盘是否被蚕食）
- **OpenAI**：无包袱，已上广告(2026.2)。
- **阿里·千问**：靠电商抽佣，agent 下单是自然延伸 → 最积极(送奶茶、ACT 协议)。
- **百度**：搜索广告被 AI 搜索蚕食 → 最纠结，先做自用 GEO 防御。
- **字节·豆包**：内容电商+信息流广告会被 agent 吃 → 克制，抽佣 2027 错位放量。
- **DeepSeek**：无第二条腿 → 不做 C 端变现，当"清流"，靠极致省+开源+影响力。
